Hiragana / ひらがな

Es un silabario diseñado para escribir palabras de origen japonés pero principalmente para escribir conjugación y partículas.

Está compuesto de 5 vocales y 41 silabas, también existen diferentes combinaciones para formar diferentes sonidos.

Saber primero el silabario Hiragana te hará más fácil aprender el Katakana.
